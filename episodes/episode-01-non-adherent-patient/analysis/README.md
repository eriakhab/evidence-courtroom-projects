# Episode 01 — Analysis Documentation

This folder contains the statistical analysis scripts used in
Episode 01: The People vs. The Non-Adherent Patient.

## Files

### pmn_analysis.sas
Primary SAS script for:
- Data preparation
- Descriptive analysis
- Regression modeling
- Model diagnostics

### pmn_analysis.sps
SPSS syntax for:
- Data cleaning
- Variable construction
- Supplementary analyses

## Data Source

All analyses use:

/data/PMN_data.sav

## Workflow

1. Load raw data from /data
2. Run preprocessing steps
3. Execute regression models
4. Generate outputs for manuscripts

## Notes

These scripts reflect analyses conducted for the
master’s thesis (2023) and subsequent manuscript (2024).
